getAllMessages:

  using GET

explane:
Not getting any parameter and returns all messages.

getMessageByID:

  using POST

explane:
Gets one parameter ("id")
and returns all messages that have spcific ID.

getMessageByDateAndID:

  using POST

explane:
Gets two parameters ("id" , "date")
and returns all messages that have spcific ID with spcific Date.
